# Guest List Manager

A simple web app to manage a guest list with RSVP toggle and removal functionality.

## Features

- Add guests (max 10)
- Toggle RSVP status (Attending / Not Attending)
- Remove guests from the list
- Prevent adding empty or duplicate names
